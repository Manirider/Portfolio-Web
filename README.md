## About
Personal portfolio website for Manikanta Suryasai Sunkara — AI/ML Engineer, Full-Stack Developer, and Blockchain Developer. Built to showcase production-grade AI systems, scalable backends, and decentralized protocols.

## Tech Stack
- **Framework:** React 18 + Vite + TypeScript
- **Styling:** Tailwind CSS v3 + Custom CSS
- **Animations:** Framer Motion + GSAP ScrollTrigger
- **Icons:** Lucide React + React Icons
- **Deployment:** Vercel

## Features
- 🎨 Custom "Neural Noir" design system (dark, cinematic, technical)
- ⚡ Framer Motion animations with prefers-reduced-motion support
- 📱 Fully responsive (375px, 768px, 1280px breakpoints)
- 🚀 Lighthouse scores: Performance 90+, Accessibility 95+, SEO 90+
- 🌐 Live deployed on Vercel with CI/CD via GitHub integration

## Local Setup
```bash
git clone https://github.com/Manirider/portfolio
cd portfolio
npm install
npm run dev
# Visit http://localhost:5173
```

## Build
```bash
npm run build
npm run preview
```
